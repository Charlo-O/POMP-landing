Privacy Policy

Pomp 隐私协议

Pomp 是一个本地优先的 Chrome 扩展，用来收集 AI 图像、提示词、来源、标签和备注。我们不会运营用于收集用户数据的服务器。

生效日期：2026 年 6 月 8 日

1. 我们不会主动收集的信息

Pomp 不会主动收集你的姓名、邮箱、账号、浏览历史、支付信息或精确位置，也不包含广告追踪、分析埋点或第三方统计 SDK。

2. 本地保存的数据

当你使用 Pomp 保存内容时，图片、提示词、反向提示词、分类、标签、备注、来源 URL 和相关设置会优先保存在你的浏览器本地，例如 IndexedDB、Chrome storage 或下载目录。

这些数据默认不会发送给 Pomp 作者或任何 Pomp 自建服务。你可以在扩展内删除记录，或通过浏览器的扩展数据管理功能清除本地数据。

3. 你主动启用的第三方服务

如果你配置并使用 OpenAI 兼容 API、GitHub、Notion 或飞书多维表格，Pomp 会把完成对应功能所需的数据发送给你选择的第三方服务。

例如，反推提示词或智能分类可能会把图片内容发送给你配置的视觉模型服务；同步功能可能会把图片、提示词和记录写入你配置的 GitHub 仓库、Notion 数据库或飞书表格。第三方服务会按照它们自己的隐私政策处理这些数据。

4. 权限用途

Pomp 使用浏览器权限来完成扩展功能，包括在网页中显示收图入口、读取你主动选择保存的页面内容、保存下载文件、打开扩展弹窗和保存本地设置。

这些权限只用于提供收集、整理、生成、导出和同步功能，不用于广告追踪或用户画像。

5. API Key 和访问令牌

你在 Pomp 中填写的 API Key、GitHub token、Notion token 或飞书访问配置会保存在你的本地浏览器中，用于向你配置的服务发起请求。请妥善保管这些凭证，并只授予必要权限。

6. 数据导出与删除

Pomp 支持将本地记录导出到你的下载目录，或同步到你配置的第三方服务。你可以在扩展中删除本地记录，也可以在对应第三方服务中删除已同步的数据。

7. 政策更新

如果 Pomp 的数据处理方式发生实质变化，我们会更新本页面，并在项目仓库中保留可追踪的变更记录。

8. 联系方式

如对隐私协议或数据处理方式有疑问，可以通过 GitHub Issues 联系项目维护者。
