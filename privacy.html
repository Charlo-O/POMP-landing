Pomp 隐私说明

Pomp 是一个本地优先的 Chrome 扩展。它的核心用途是帮你保存 AI 图像、提示词、来源、标签和备注；默认没有 Pomp 自有服务器，也不会把你的图库回传给作者。

最后更新：2026 年 6 月 8 日

默认本地 记录保存在浏览器 IndexedDB 和本地扩展存储中。

不卖数据 不做广告画像，不出售数据，不把图库交给数据经纪商。

适用范围

本说明适用于 Pomp Chrome 扩展，以及这个静态落地页。落地页不包含统计脚本、广告脚本或登录系统；扩展中的数据处理发生在你的浏览器、你配置的 API 服务，以及你主动启用的同步目标之间。

我们处理哪些信息

图库记录

你保存的图片数据或图片地址、来源页面 URL、图片来源 URL、正向提示词、负向提示词、反推提示词、分类、标签、备注、创建和更新时间、生成来源以及同步状态。

页面上下文

当你右键保存图片、点击页面浮标、收集选中文本或补充提示词时，Pomp 会读取当前页面 URL、图片 URL、图片 alt/title、选中文本、图片附近文本和页面元信息，用来提取 prompt 候选、来源和标签。近期采集上下文可能会短暂写入本地扩展存储，以便把刚保存的图片和后续补充的提示词接起来。

设置与凭证

API 地址、API Key、模型名称、系统提示词、下载目录、允许显示浮标的网站、标签面板、采集模板，以及你填写的 GitHub Token、仓库信息、Notion Token、Database ID、飞书 Access Token、App Token 和 Table ID。

AI 处理结果

反推得到的提示词、自动分类结果、标签建议、备注建议、生成图结果、生成图修订提示词，以及与这些结果相关的错误状态。

这些信息如何使用

  • 保存、搜索、筛选和导出你的 AI 图片提示词图库。
  • 在你触发反推、分类或生图时，把必要的图片或提示词发送给你配置的 OpenAI-compatible API。
  • 在你启用自动导出或点击导出时，把记录写入浏览器下载目录。
  • 在你启用远程同步或手动同步时，把必要字段写入你配置的 GitHub、Notion 或飞书目标。
  • 保存扩展 UI 状态，例如浮标位置、当前采集流程和最近生成状态。

数据保存在哪里

位置 保存内容
浏览器 IndexedDB 图库记录和图片 data URL，数据库名为 pomp-ai-prompt-archive。
Chrome 本地扩展存储 设置、API Key、同步 Token、待处理采集上下文和 UI 状态。
浏览器下载目录 你主动导出的 ZIP、HTML 表格、图片、prompt 文本和 JSON 记录。
你启用的第三方服务 只有在你配置并启用对应同步或 API 功能时才会写入。

会和哪些第三方共享

Pomp 没有自有云端数据库。下列共享都由你主动配置或触发，且用于完成你在扩展里看得到的功能。

OpenAI-compatible API

默认地址是 OpenAI API。你也可以改成其他兼容端点。反推、分类和生图时，相关图片、图片 URL、prompt、负向 prompt 和设置里的系统提示词会发送到该端点。

GitHub

启用后，Pomp 会把图片、prompt 文本、负向 prompt、来源 URL、单条记录 JSON 和 records.json 表写入你指定的仓库路径。公开仓库会让这些内容公开可见。

Notion

启用后，Pomp 会向你指定的 Notion Database 创建页面，写入 prompt、分类、标签、来源 URL、图片 URL、备注，以及负向 prompt 和反推 prompt 文本。

飞书多维表格

启用后，Pomp 会向你指定的多维表格写入 prompt、负向 prompt、分类、标签、来源 URL、图片 URL 和备注字段。

Chrome 权限说明

storage 保存扩展设置、采集状态和图库相关本地数据。
downloads 把 ZIP、HTML 表格、图片和记录文件导出到浏览器下载目录。
contextMenus 提供右键保存图片、反推图片、保存选中文本等入口。
activeTab 让扩展在你当前操作的标签页完成采集、提示和状态更新。
<all_urls> 在用户访问的网站上注入 Pomp 内容脚本，以显示浮标、识别图片、读取来源和提取图片附近的 prompt。浮标默认只在设置里的允许网站显示。

安全与限制

API Key 和同步 Token 保存在 Chrome 本地扩展存储中，Pomp 不会把它们发送给作者的服务器。请只在可信设备和可信 Chrome 配置文件里使用这些凭证。默认 API 和内置同步端点使用 HTTPS；如果你自定义 API 地址，请优先使用 HTTPS 端点。

你的控制

  • 你可以在 Pomp 弹窗里删除单条图库记录。
  • 你可以在设置页删除或替换 API Key、Token、同步目标和允许显示的网站。
  • 你可以关闭自动下载和自动同步；不开启远程目标时，记录不会同步到 GitHub、Notion 或飞书。
  • 卸载扩展或清除扩展站点数据会移除浏览器内的本地扩展数据。已经导出到下载目录、GitHub、Notion 或飞书的数据，需要到对应位置自行删除。

Chrome Web Store Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Pomp 仅将通过 Chrome 扩展权限取得的信息用于提供或改进“保存 AI 图片提示词图库”这一用户可见功能。Pomp 不会把用户数据用于个性化广告、再营销、信用评估，也不会出售给广告平台、数据经纪商或信息转售方。

儿童隐私

Pomp 面向需要整理 AI 图像工作流的创作者和研究者，不面向儿童提供服务，也不会有意收集儿童个人信息。

变更

如果扩展的数据处理方式发生实质变化，本说明会随项目更新。发布到 Chrome Web Store 时，应确保商店后台披露、扩展 UI 和本说明保持一致。

联系

你可以通过 GitHub Issues 反馈隐私、安全或数据处理问题。
